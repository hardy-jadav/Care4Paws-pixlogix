# Care4Paws — Pet Care Services Website

A fully responsive single-page marketing website for a professional pet care company offering grooming, boarding, training, and veterinary services. Built with HTML5, Tailwind CSS, and JavaScript, powered by Vite.

---

## 🔗 Live Demo

👉 https://care4-paws-pixlogix.vercel.app/

---

## 🚀 Tech Stack

- HTML5 — Semantic structure  
- Tailwind CSS — Styling  
- JavaScript (ES6+) — Interactivity  
- Vite — Build tool  
- PostCSS + Autoprefixer — CSS processing  
- Font Awesome — Icons  
- Google Fonts (Montserrat) — Typography  

---

## ⚙️ Getting Started

### Prerequisites
- Node.js 18+
- npm 9+

### Installation

```bash
git clone <repository-url>
cd Project
npm install
```

### Development

```bash
npm run dev
```

Runs on: http://localhost:5173

### Production Build

```bash
npm run build
```

### Preview Build

```bash
npm run preview
```
